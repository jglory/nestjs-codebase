# 시작하기


문서위치 : [README](../README.md)



## 설치

```bash
$ yarn install

# set env
$ cp .env.sample .env
```

## 실행

```bash
# db migration
$ yarn run migration:run

# development
$ yarn start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## 테스트

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
